# Desktop-Organizer
python script to keep desktop organized.
this script will move files of same type into a folder,this is done for all types of files keeping your desktop organized
Modules used in this project are shutil and os
first the desktop path is extracted,then every file in desktop is given a path then the files are segregated using a list within a dictionary then they are moved into seperate folders made for different types of files using shutil module
